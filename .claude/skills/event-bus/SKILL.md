# Event Bus

## Overview

Socket hooks dispatch typed `CustomEvent`s on `window`. UI components subscribe via `window.addEventListener` (wrapped in a `subscribe*` helper). This decouples the Socket.io layer from React state — socket hooks never touch React state directly. Each domain has its own event module in `src/lib/`.

## Implementation Pattern

```ts
// src/lib/myFeatureEvents.ts

// 1. Define payload interfaces
export interface IncomingMyEvent {
  id: number;
  title: string;
  actorName: string;
}

// 2. Scoped event name constants (never export raw strings)
const MY_FEATURE_CREATED = "noctification:my-feature:created";
const MY_FEATURE_UPDATED = "noctification:my-feature:updated";

// 3. Dispatch helpers (called from socket hooks)
export const dispatchMyFeatureCreated = (payload: IncomingMyEvent) => {
  window.dispatchEvent(
    new CustomEvent<IncomingMyEvent>(MY_FEATURE_CREATED, { detail: payload })
  );
};

export const dispatchMyFeatureUpdated = (payload: IncomingMyEvent) => {
  window.dispatchEvent(
    new CustomEvent<IncomingMyEvent>(MY_FEATURE_UPDATED, { detail: payload })
  );
};

// 4. Subscribe helper (called from UI components / custom hooks)
export const subscribeMyFeatureEvents = (callbacks: {
  onCreated: (payload: IncomingMyEvent) => void;
  onUpdated?: (payload: IncomingMyEvent) => void;
}) => {
  const handleCreated = (event: Event) => {
    callbacks.onCreated((event as CustomEvent<IncomingMyEvent>).detail);
  };
  const handleUpdated = (event: Event) => {
    callbacks.onUpdated?.((event as CustomEvent<IncomingMyEvent>).detail);
  };

  window.addEventListener(MY_FEATURE_CREATED, handleCreated);
  window.addEventListener(MY_FEATURE_UPDATED, handleUpdated);

  // Return cleanup
  return () => {
    window.removeEventListener(MY_FEATURE_CREATED, handleCreated);
    window.removeEventListener(MY_FEATURE_UPDATED, handleUpdated);
  };
};
```

## Usage in Component

```ts
useEffect(() => {
  const cleanup = subscribeMyFeatureEvents({
    onCreated: (payload) => {
      setItems((prev) => [toLocalItem(payload), ...prev]);
    }
  });
  return cleanup;
}, []);
```

## Rules

### Do
- Prefix all event name constants with `noctification:` to avoid collisions
- Keep event name constants private (not exported) — expose only helpers
- Return the cleanup function from `subscribe*` helpers
- Always `removeEventListener` with the same function reference used in `addEventListener`

### Don't
- Don't dispatch events directly from components — only from socket hooks
- Don't use `eventTarget.dispatchEvent` on DOM nodes — always `window`
- Don't share event name strings across modules — define per-module
