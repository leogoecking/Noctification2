# UI Constants Module

## Overview

Each complex feature has a co-located `{feature}Ui.ts` (never `.tsx`) that exports:
- `Record<DomainEnum, string>` label maps
- Palette / badge style objects (Tailwind strings or inline style objects)
- `format*` date/number formatters using `Intl`
- Derived types and helper functions used only by the feature's components

This keeps components thin and makes the display layer independently testable.

## File Structure

```
src/components/{feature}/
├── {Feature}.tsx         # component — imports from Ui module
├── {feature}Ui.ts        # NO JSX — pure TS: constants, types, helpers
└── {Feature}.test.tsx
```

## Implementation Pattern

```ts
// src/components/my-feature/myFeatureUi.ts

import type { MyStatus, MyPriority } from "../../types";

// 1. Label maps (Record ensures exhaustive coverage)
export const MY_STATUS_LABELS: Record<MyStatus, string> = {
  new: "Nova",
  active: "Ativa",
  done: "Concluída"
};

// 2. Badge Tailwind classes
export const MY_STATUS_BADGES: Record<MyStatus, string> = {
  new: "bg-accent/10 text-accent",
  active: "bg-warning/10 text-warning",
  done: "bg-success/10 text-success"
};

// 3. Palette for colored cards (dark mode aware)
export type MyPalette = {
  bg: string;
  border: string;
  titleColor: string;
};

export const MY_PALETTES: Record<MyPriority, { light: MyPalette; dark: MyPalette }> = {
  normal: {
    light: { bg: "#E3F2FD", border: "#90CAF9", titleColor: "#0D2B4E" },
    dark:  { bg: "#051525", border: "#0e4060", titleColor: "#cdf0ff" }
  },
  critical: {
    light: { bg: "#FCE4EC", border: "#F48FB1", titleColor: "#4A0020" },
    dark:  { bg: "#1a0508", border: "#7a1a20", titleColor: "#ffe8ea" }
  }
};

export const getMyPalette = (priority: MyPriority, darkMode: boolean): MyPalette =>
  darkMode ? MY_PALETTES[priority].dark : MY_PALETTES[priority].light;

// 4. Formatters use Intl, never raw Date methods
export const formatMyDate = (value: string | null): string => {
  if (!value) return "-";
  return new Intl.DateTimeFormat("pt-BR", {
    day: "2-digit",
    month: "2-digit",
    year: "numeric",
    hour: "2-digit",
    minute: "2-digit"
  }).format(new Date(value));
};

// 5. isDarkModeActive helper (copy per module, avoids circular dep)
export const isDarkModeActive = (): boolean =>
  typeof document !== "undefined" && document.documentElement.classList.contains("dark");
```

## Rules

### Do
- Use `Record<DomainEnum, V>` — TypeScript will catch missing keys when the enum grows
- Use `Intl.DateTimeFormat` / `Intl.NumberFormat` for all formatting
- Provide both `light` and `dark` palettes when using inline styles
- Export `isDarkModeActive()` per module to avoid coupling

### Don't
- Don't put JSX in `*Ui.ts` files — keep extension `.ts`, not `.tsx`
- Don't import from the component into the Ui module (circular dep)
- Don't hardcode hex colors in components — put them in the Ui module

## File Location

`src/components/{feature}/{feature}Ui.ts`  
`src/features/{domain}/components/{feature}Ui.ts`
