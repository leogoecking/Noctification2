# Custom Hook

## Overview

Hooks accept a single options object (never positional params). Side effects are cleaned up in `useEffect` return functions. Hooks that manage persistent preferences use `localStorage` with an SSR guard (`typeof window === "undefined"`). Async data-loading hooks expose `loading` state and call `onError(string)` on failure.

## Implementation Pattern

### UI Behaviour Hook
```ts
// src/hooks/useHoverExpandableSidebar.ts
interface UseHoverExpandableSidebarOptions {
  storageKey: string;
  defaultPinned?: boolean;
  collapseDelayMs?: number;
}

export const useHoverExpandableSidebar = ({
  storageKey,
  defaultPinned = false,
  collapseDelayMs = 120
}: UseHoverExpandableSidebarOptions) => {
  const [isPinned, setIsPinned] = useState<boolean>(() =>
    getInitialPinnedState(storageKey, defaultPinned)
  );
  const [isHovered, setIsHovered] = useState(false);
  const collapseTimeoutRef = useRef<number | null>(null);

  // Persist preference
  useEffect(() => {
    window.localStorage.setItem(storageKey, isPinned ? "true" : "false");
  }, [isPinned, storageKey]);

  // Cleanup timeout on unmount
  useEffect(() => {
    return () => {
      if (collapseTimeoutRef.current !== null) {
        window.clearTimeout(collapseTimeoutRef.current);
      }
    };
  }, []);

  return { isExpanded: isPinned || isHovered, isPinned, onMouseEnter, onMouseLeave, togglePinned };
};
```

### Data Loading Hook
```ts
// pattern from useAdminDashboardData.ts
interface UseMyDataOptions {
  onError: (message: string) => void;
}

export const useMyData = ({ onError }: UseMyDataOptions) => {
  const [data, setData] = useState<MyItem[]>([]);
  const [loading, setLoading] = useState(false);
  const mountedRef = useRef(true);

  useEffect(() => {
    mountedRef.current = true;
    return () => { mountedRef.current = false; };
  }, []);

  const load = useCallback(async () => {
    setLoading(true);
    try {
      const response = await api.myItems();
      if (mountedRef.current) setData(response.items);
    } catch (error) {
      if (mountedRef.current) {
        onError(error instanceof ApiError ? error.message : "Falha ao carregar");
      }
    } finally {
      if (mountedRef.current) setLoading(false);
    }
  }, [onError]);

  useEffect(() => { void load(); }, [load]);

  return { data, loading, refresh: load };
};
```

## Rules

### Do
- Accept a single options object, never positional arguments
- Guard `setState` calls with `mountedRef.current` in async callbacks
- Clean up timers, subscriptions and sockets in the `useEffect` return
- Wrap async loaders in `useCallback` with proper dependency arrays
- Guard `localStorage` / `window` access with `typeof window === "undefined"`
- Return a plain object (not array) unless the hook mimics `useState`

### Don't
- Don't `throw` from hooks — call `onError(string)` instead
- Don't skip cleanup of `setTimeout` / `setInterval` refs
- Don't read stale closure state — use refs for mutable counters (`loadRequestIdRef`)
