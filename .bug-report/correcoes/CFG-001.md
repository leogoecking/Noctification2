# CFG-001

## Resumo do problema

O script `npm test` da raiz do monorepo executava apenas a suite da API, deixando o workspace web fora da validacao padrao.

## Causa raiz

O script `test` em `package.json` apontava diretamente para `@noctification/api`.

## Arquivos alterados

- `package.json`

## Estrategia da correcao

- Fazer o script raiz encadear `test:api` e `test:web`.
- Preservar os scripts separados existentes para execucao direcionada.

## Riscos considerados

- Baixo risco funcional.
- A principal mudanca e aumento de cobertura e tempo de execucao do teste raiz.

## Validacao executada

- `npm test`
- confirmacao de execucao da API e do Web no mesmo fluxo
