# REF-002

## Resumo do problema

`AdminDashboard.tsx` concentrava busca global, health check e composição de múltiplos subdomínios administrativos.

## Causa raiz

- lógica de busca global embutida no componente container;
- health check administrativo também embutido no mesmo arquivo.

## Arquivos alterados

- `apps/web/src/components/AdminDashboard.tsx`
- `apps/web/src/components/admin/useAdminGlobalSearch.ts`
- `apps/web/src/components/admin/useAdminSystemHealth.ts`

## Estratégia da correção

- extrair busca global para hook dedicado;
- extrair health check administrativo para hook dedicado;
- manter o container responsável por composição, não por efeitos secundários detalhados.

## Riscos considerados

- regressão em testes do dashboard;
- mudança sutil no momento de atualização da busca.

## Validação executada

- `npm run test --workspace=apps/web -- AdminDashboard`
- `npm run test --workspace=apps/web`
- `npm run typecheck --workspace @noctification/web`
