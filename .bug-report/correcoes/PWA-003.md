# PWA-003

## Resumo

Fechamento da configuracao local restante: geracao de chaves VAPID, ajuste do `.env` local da API para `https://noctification.lan` e script de geracao de certificados locais via `openssl`.

## Arquivos alterados

- `apps/api/.env`
- `ops/scripts/generate-local-certs.sh`
- `README.md`

## Estrategia

- Gerar VAPID localmente com a dependencia `web-push`
- Ajustar a API local para o hostname HTTPS da rede
- Automatizar a emissao de CA local e certificado do servidor sem depender de `mkcert`

## Riscos

- As chaves VAPID foram geradas localmente para este ambiente; se quiser rotacionar, basta gerar novas e atualizar os envs.
- O HTTPS so sera confiavel nos dispositivos apos importar a CA raiz gerada.

## Validacao executada

- `npm run generate-vapid --workspace @noctification/api`
- leitura do `.env` local apos atualizacao
