# REF-001

## Resumo do problema

`App.tsx` concentrava navegação, bootstrap de sessão, tema e fila de toasts.

## Causa raiz

- ausência de hooks específicos para responsabilidades transversais da aplicação.

## Arquivos alterados

- `apps/web/src/App.tsx`
- `apps/web/src/hooks/useAppNavigation.ts`
- `apps/web/src/hooks/useThemePreference.ts`
- `apps/web/src/hooks/useToastQueue.ts`
- `apps/web/src/hooks/useSessionBootstrap.ts`

## Estratégia da correção

- extrair navegação para hook próprio;
- extrair preferência de tema;
- extrair fila de toasts;
- extrair bootstrap de sessão.

## Riscos considerados

- regressão no fluxo inicial de sessão;
- erro de sincronização entre rota atual e renderização das telas.

## Validação executada

- `npm run test --workspace=apps/web -- App`
- `npm run test --workspace=apps/web`
- `npm run typecheck --workspace @noctification/web`
